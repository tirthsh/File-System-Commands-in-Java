story 2: As a user, I should be able to exit the program whenever I want. 1 day
story 3: As a user, I should be able to create directories which may be relative to the current directory. 1 day
story 4: As a user, I should be able to create directories relative to a full path. 1 day

story 6: As a user, I should be able to change my current working directory using a full path. 1 day

story 9: As a user, I should be able to view my current working directory. 1 day
